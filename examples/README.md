# Examples

This directory contains domain-specific SPECULA examples.

## Available Examples

| Example | Domain | Real / Fictional | Key feature |
|---|---|---|---|
| [`community-space-brand/`](./community-space-brand/) | Social innovation | **Real** | Brand values from co-design → governance rules via Specula Method |

**Start with `community-space-brand`** if you want to understand how values become governance. Start with `luxury-fashion-brand` if you want a minimal structural example.

Each example includes:

- explanatory notes
- validation output
